Errors Changed initial writing: 2004-05-07
last updated: 2004-05-20

The following changes have been made:

  • Appendix D has been added to the dictionary on the errors that can be signalled by the system.
  • valence error has been replaced by domain error.
  • Errors that can never be signalled have been removed from the interpreter tables:
     defn error
     not reassignable
     result error
