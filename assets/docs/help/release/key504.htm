#/. Improved initial writing: 2005-01-20
last updated: 2005-01-21

The verb x#/.y is now faster and leaner on boolean list arguments.

   ts=: 6!:2 , 7!:2@]
   x=: 1e6 ?@$ 2

   ts '#/.~ x'    NB. J 5.03
0.00201049 1024
   ts '#/.~ x'    NB. J 5.04
0.000393657 960

   0.00201049 1024 % 0.000393657 960 
5.10721 1.06667
