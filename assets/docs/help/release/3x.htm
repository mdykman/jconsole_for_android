3!:n Changed initial writing: 2005-12-14
last updated: 2005-12-19

3!:1 binary representation and 3!:3 hex representation have been changed to facilitate its decoding by 3!:2 and other programs. See the dictionary for details.

Binary and hex representations produced by J versions prior to J6.01 can be decoded by 3!:2 in J6.01, but not vice versa.
