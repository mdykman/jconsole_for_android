editline History

Unix jconsole uses editline and identifies itself as 'jconsole'. The input history from previous sessions is loaded from file ~/.jhistory (if it exists) when jconsole starts and the new input lines are added to the file on exit. Create an empty ~/.jhistory file to make use of this feature.
