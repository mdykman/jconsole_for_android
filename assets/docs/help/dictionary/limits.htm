Appendix C. System Conventions and Limits

error number    (right argument of 13!:8) less than 256
print precision less than or equal to 20
tolerance less than or equal to 2^_34
recursion limit (function call depth limit) 100 for Windows CE, 10000 for all others

The value part of arrays with internal type boolean, literal, or Unicode, has an extra terminating byte of 0.
