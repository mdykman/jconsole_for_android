q: Improved initial writing: 2006-07-12
last updated: 2006-07-12

The dyad x q: y where x is a small non-negative integer (up to 1229) and y is a large extended integer, now performs much better — faster, leaner, and producing a result when previously it might have signalled error. For example:

   y=: 60 * 1+2^300x
   5 q: y
2 1 1 0 0

   ts=: 6!:2 , 7!:2@]
   ts '5 q: y'
1.45417e_5 4032
