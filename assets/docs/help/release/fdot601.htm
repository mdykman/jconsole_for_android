f. Improved initial writing: 2005-08-19
last updated: 2005-11-21

u f. has been improved so that a verb using $: can be fixed. For example: 

   fact=: 1: ` (* $:@<:) @. *

   >:@fact f.
>:@(3 : '1:`(* $:@<:)@.* y.' :(4 : 'x. 1:`(* $:@<:)@.* y.'))

   fact@>: f.
3 : '1:`(* $:@<:)@.* y.'@>:
