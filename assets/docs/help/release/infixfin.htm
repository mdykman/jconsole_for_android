u/\ Improved initial writing: 2003-12-04
last updated: 2005-01-20

The dyad m u/\ y is now implemented by special code for the cases where m is a positive integer and:

+ /\  boolean, integer or floating point y
>./\ boolean, integer or floating point y
<./\ boolean, integer or floating point y
+./\  boolean y
*./\  boolean y
= /\  boolean y
~:/\  boolean y
