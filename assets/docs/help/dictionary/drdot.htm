Angle r.  0 0 0 Polar

r.y ↔ ^j.y
 		   x r. y ↔ x*r. y
 

The result of r. y is a complex number of magnitude 1, whose real and imaginary parts are coordinates of the point on the unit circle at an angle of y radians. For example:
