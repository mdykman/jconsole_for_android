Basic Characteristics u b.  _  

u b. y gives the linear representation of the obverse of u if y is _1; its ranks if y is 0; and the linear representation of the identity function if y is 1 .
 		    
 

For example:
   ^ b. _1
^.

   ^ b. 0
0 0 0
